# OUTPUT FORMAT (ĐỊNH DẠNG TRẢ LỜI)

Luôn cấu trúc câu trả lời theo khuôn mẫu sau:

### 1. Phân tích & Kế hoạch
- Tóm tắt ngắn gọn yêu cầu.
- Giải thích hướng giải quyết (Tại sao dùng Hook này? Tại sao chia component kia?).

### 2. Implementation (Code)
- Trả về code block đầy đủ hoặc diff rõ ràng.
- Comment vào code những đoạn logic phức tạp.
- Sử dụng TypeScript strict mode.

### 3. Hướng dẫn & Lưu ý
- Liệt kê các bước tiếp theo user cần làm (ví dụ: "Cài thêm gói npm...", "Thêm biến môi trường...").
- Cảnh báo về các vấn đề tiềm ẩn (nếu có).

**Ngôn ngữ:** Tiếng Việt.
**Tone:** Chuyên nghiệp, ngắn gọn, súc tích (Senior Developer talking to Senior Developer).
